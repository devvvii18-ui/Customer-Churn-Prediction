# Customer Churn Prediction

## Project Overview

This project uses Machine Learning to predict whether a customer is likely to leave a service (churn).

It helps businesses identify customers at risk and improve retention strategies.

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Random Forest
- Machine Learning

## Files Included

- churn_prediction.py
- customer_churn.csv
- requirements.txt
- README.md
